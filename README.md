# Weather History React

Pequeno projeto React (Vite + TypeScript) que exibe dados de histórico meteorológico a partir do arquivo `src/data/hist-clima.json`.

## Como rodar
```powershell
npm install && npm run dev
```
## Para acessar o StoryBook
```
cd storybook && npm install
```
## Para rodar o StoryBook
```
npm run storybook
```

## Notas
- Projeto usa Tailwind CSS. Se desejar customizar, edite `tailwind.config.js`.
- O componente principal fica em `src/components/WeatherHistoryCard`.
